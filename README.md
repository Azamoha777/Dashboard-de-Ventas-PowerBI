# Dashboard-de-Ventas-PowerBI

## Descripción del proyecto

Proyecto de tablero interactivo sobre ventas desarrollado en Microsoft Power BI. Este dashboard permite analizar el desempeño comercial mediante indicadores clave y visualizaciones dinámicas que facilitan la interpretación de datos y la toma de decisiones dentro de la empresa.

El proyecto integra diferentes perspectivas del negocio, incluyendo análisis de ventas, comportamiento de productos, distribución geográfica, desempeño logístico e información de clientes.

---

# Objetivo

Analizar las ventas totales, el comportamiento por producto, país y fecha, así como evaluar el rendimiento de entregas para detectar oportunidades de mejora en la operación comercial y logística.

---

# Herramientas utilizadas

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)
- Modelado de datos
- Visualización de datos

---

# Indicadores principales

- Ventas totales
- Ventas por país
- Ventas por año y mes
- Productos más vendidos
- Tiempo de entrega
- Información de clientes

---

# Análisis del dashboard

## Ventas por Producto

Se presenta un análisis de los productos con mayores ventas mediante gráficos comparativos, permitiendo identificar cuáles generan mayor ingreso para la empresa y cuáles tienen mejor desempeño comercial.

## Ventas por Año

El dashboard analiza la evolución de las ventas por año y por mes, facilitando la identificación de tendencias de crecimiento, temporadas de mayor demanda y variaciones en el comportamiento comercial.

## Ventas por País

Se incluye un mapa interactivo que permite visualizar la distribución geográfica de las ventas, identificando los países con mayor participación dentro del mercado.

## Tiempo de Entrega

Se evalúan los días de entrega de los pedidos para analizar el desempeño logístico y detectar posibles retrasos que puedan afectar la satisfacción del cliente.

## Información de Clientes

El tablero integra información relevante de clientes, como nombre de empresa, contacto y teléfono, complementando el análisis comercial y permitiendo una mejor organización de la información empresarial.
